# bdsmgr
Minecraft 基岩版服务端（BDS）管理脚本

（适用于 Termux）

## 功能
- 安装 BDS
- 更新 BDS
- 卸载 BDS

## 使用方法
在 Termux 中执行以下命令

```
wget -qO bds.sh https://raw.gitmirror.com/hmsjy2017/bdsmgr/main/bds.sh && bash bds.sh
```
